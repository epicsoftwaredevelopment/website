Privacy policy

Ad-supported businesses often track a lot of information about you to better serve advertisers. Multiplayer Match isnâ€™t like that. We have no ads and the game was not created to make money. We only collect enough information to run Multiplayer Match.

Login

If you decide to register with a phone number we store this information in a database on the Multiplayer Match servers. Multiplayer Match does not have traditional passwords. Instead, when you login we generate a random 6 digit code to establish your session. We do not use this information for any other purpose.

Supporting services

Multiplayer Match uses Twilio for SMS to send you the 6 digit code mentioned above. Because they send the text message, they have a record in a database with your phone number. You can read more about the Twilio privacy policy here.
